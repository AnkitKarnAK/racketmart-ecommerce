# RacketMart E-commerce web app

## Live link: [RacketMart](https://racketmart.netlify.app/)

## Features:

1. Auth (Login & Signup)
2. Private Routes
3. Add to Cart
4. Add to Wishlist
5. Increase or Decrease cart quantity
